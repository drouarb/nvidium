# Nvidium

[![Modrinth](https://img.shields.io/modrinth/dt/nvidium?logo=modrinth)](https://modrinth.com/mod/nvidium)

Nvidium is an alternate rendering backing for [**Sodium**](https://modrinth.com/mod/sodium), it uses cutting edge [**NVIDIA**](https://www.nvidia.com) features to render huge amounts of terrain geometry at very playable framerates!

### Requires [Sodium](https://modrinth.com/mod/sodium) and an [NVIDIA GTX 1600 series](https://www.nvidia.com/en-us/geforce/graphics-cards/16-series/) or newer to run (Turing+ architecture).
